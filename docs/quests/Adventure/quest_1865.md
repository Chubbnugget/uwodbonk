### Quest Contents
Requests have come from European naturalists. Apparently there is a tree in the south that is said to have the effect that ``travelers who eat its fruit will return to that place again.'' Details seems to want to know what kind of tree it is. Maybe you can find it if you go south as far as Palparaiso.
