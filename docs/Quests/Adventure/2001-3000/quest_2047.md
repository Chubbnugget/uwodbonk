### Quest Contents
This is a survey about the strait that we pass through to get to Kaffa. It seems that a kingdom called Bosporus flourished in that area a long time ago. A historian is on his way to investigate, so he wants you to come and see what the strait looks like. It seems like it's a pretty narrow place, so make sure you don't run aground!
