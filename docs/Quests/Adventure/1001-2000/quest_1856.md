### Quest Contents
This is a request that can only be requested by voyagers who have reached Machu Picchu. Apparently there is a very sharp mountain next to Machu Picchu. This time, please gather information about the mountain near Machu Picchu. First of all, could you please talk to the gatekeeper in Lima?
