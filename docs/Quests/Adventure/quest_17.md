### Quest Contents
Anatolia is famous for Hittite iron, but since iron corrodes easily, there isn't much left. But if we study the bronze tools used before the Hittites, wouldn't we be able to understand how the Hittites came to make iron? So, here's a request. First, talk to the client's scholar.
