### Quest Contents
A pioneer group from Europe is coming to Lambayeque. It seems like they're planning to build a cultivation base like they did here, but... it doesn't seem to be going well. The pioneering group has received a request to deliver 100 barrels of ham, sausage, and bacon all at the same time. You seem to be in a lot of trouble...
