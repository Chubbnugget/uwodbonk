### Quest Contents
This is a request to recruit instructors for graduates of a training school's commercial course. You have the skills needed to be an instructor. If you want to become a commerce instructor, gather knowledge useful for trade from the merchant guild masters in each home base, compile it into a paper, and submit it to the merchant guild master in Seville. If you can submit it, we will issue you a license.
