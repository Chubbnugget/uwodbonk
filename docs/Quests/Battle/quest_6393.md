### Quest Contents
This is a request to recruit instructors for graduates of the maritime course at a navigator training school. You have the skills needed to be an instructor. If you want to become a maritime instructor, gather useful maritime knowledge from the maritime guild masters of each home base, compile it into a paper, and give it to the maritime course instructor in Seville. If you can submit it, we will issue you a license.
